# 💳 CreditWise Loan System


# 📌 Project Overview

CreditWise Loan System is a data analysis project designed to evaluate customer financial data and assess loan eligibility and risk. The project focuses on analyzing applicant details, financial status, and behavioral patterns to generate meaningful insights for better loan decision-making.

This project demonstrates practical implementation of data preprocessing, feature engineering, and exploratory data analysis (EDA) using Python.

# 🎯 Objectives

1. Analyze customer financial and personal data.

2. Clean and preprocess raw loan dataset.

3. Identify key factors affecting loan approval.

4. Perform exploratory data analysis to detect patterns.

5. Prepare data for future machine learning models.

# 📂 Dataset Information

The dataset includes information such as:

1. Applicant Income

2. Co-applicant Income

3. Loan Amount

4. Loan Term

5. Credit History

6. Gender, Education, Self-Employed Status

7. Property Area

8. Loan Status (Approved/Not Approved)
